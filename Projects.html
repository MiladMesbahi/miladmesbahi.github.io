Technical Projects

Autonomous Rover

January 2021 - March 2021

In my first hands-on robotics venture, I designed and built an autonomous rover capable of navigating an obstacle course. Utilizing SolidWorks, I developed a robust chassis to house all electronic components, motors, and sensors efficiently. A key feature was the innovative grabber mechanism equipped with rubberized clampers for object retrieval. The project extensively employed 3D printing technology, utilizing Prusa slicer software and UCLA's facilities for creating parts like motor mounts and battery pack holders. I also spearheaded the programming aspect, optimizing Arduino code to enable intelligent navigation and decision-making for the rover. This project highlights my skills in mechanical design, 3D modeling, rapid prototyping, and programming within the interdisciplinary field of robotics.
